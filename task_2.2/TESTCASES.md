# Тест-кейсы

## Десктопная версия

### TC-1: Главная страница - Фильтр «Диапазон цен»

Предусловия: Открыта главная страница, боковая панель фильтров видна.

| # | Шаг | Ожидаемый результат |

| 1 | Ввести в поле «От» значение 10000 | Поле принимает числовое значение |
| 2 | Ввести в поле «До» значение 30000 | Поле принимает числовое значение |
| 3 | Дождаться обновления списка объявлений | Отображаются только объявления с ценой в диапазоне 10 000–30 000 ₽ |
| 4 | Проверить цены на всех отображённых карточках | Каждая цена >= 10 000 и <= 30 000 |
| 5 | Очистить оба поля | Отображаются все объявления без ценового фильтра |



### TC-2: Главная страница - Сортировка «По цене»

Предусловия: Открыта главная страница, фильтры сброшены.

| # | Шаг | Ожидаемый результат |

| 1 | В выпадающем списке «Сортировать по» выбрать «Цене» | Список объявлений пересортирован |
| 2 | Убедиться, что порядок «По убыванию» | Цены идут от большей к меньшей |
| 3 | Проверить цены на текущей странице | Каждая следующая цена <= предыдущей |
| 4 | Переключить порядок на «По возрастанию» | Цены идут от меньшей к большей |
| 5 | Проверить цены на текущей странице | Каждая следующая цена >= предыдущей |



### TC-3: Главная страница - Фильтр «Категория»

Предусловия: Открыта главная страница, боковая панель фильтров видна.

| # | Шаг | Ожидаемый результат |

| 1 | В выпадающем списке «Категория» выбрать «Электроника» | Список объявлений обновляется |
| 2 | Проверить категории на всех отображённых карточках | Каждая карточка имеет категорию «Электроника» |
| 3 | Выбрать другую категорию, например «Транспорт» | Список обновляется |
| 4 | Проверить категории на всех карточках | Каждая карточка имеет категорию «Транспорт» |
| 5 | Выбрать «Все категории» | Отображаются объявления всех категорий |



### TC-4: Главная страница - Тогл «Только срочные»

Предусловия: Открыта главная страница.

| # | Шаг | Ожидаемый результат |

| 1 | Активировать тогл «Только срочные» | Тогл переключается в активное состояние |
| 2 | Дождаться обновления списка | Отображаются только объявления с приоритетом «Срочно» |
| 3 | Проверить, что каждая карточка имеет бейдж «Срочно» | Все карточки помечены как срочные |
| 4 | Проверить, что количество объявлений уменьшилось | Общее число объявлений меньше, чем до фильтрации |
| 5 | Деактивировать тогл | Отображаются все объявления |



### TC-5: Страница статистики - Контейнер управления таймером обновления

Предусловия: Открыта страница статистики через навигацию, статистика загружена.

| # | Шаг | Ожидаемый результат |

| 1 | Убедиться, что таймер отображается и ведёт обратный отсчёт | Таймер показывает время в формате M:SS, значение уменьшается |
| 2 | Нажать кнопку «Обновить» | Статистика обновляется (данные перезагружаются), таймер сбрасывается |
| 3 | Нажать кнопку паузы (⏸) таймера | Таймер останавливается, отображается текст «Автообновление выключено», кнопка меняется на ▶️ |
| 4 | Подождать 2–3 секунды | Таймер не изменяется (остановлен) |
| 5 | Нажать кнопку запуска (▶️) таймера | Таймер возобновляет обратный отсчёт, кнопка меняется на ⏸ |



## Мобильная версия

### TC-6: Переключение между тёмной и светлой темой

Предусловия: Открыта любая страница на мобильном устройстве (viewport 375x667).

| # | Шаг | Ожидаемый результат |

| 1 | Убедиться, что включена светлая тема | `data-theme="light"`, фон белый (`rgb(255, 255, 255)`), кнопка показывает «🌙 Темная» |
| 2 | Нажать кнопку переключения темы | Тема переключается на тёмную |
| 3 | Проверить тёмную тему | `data-theme="dark"`, фон тёмный (`rgb(26, 26, 26)`), кнопка показывает «☀️ Светлая» |
| 4 | Нажать кнопку переключения темы ещё раз | Тема переключается обратно на светлую |
| 5 | Проверить светлую тему | `data-theme="light"`, фон белый, кнопка показывает «🌙 Темная» |
